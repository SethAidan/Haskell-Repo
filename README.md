# Small Haskell Scripts
The purpose of this repository is to hold a variety of smaller Haskell programs that I have written at the beginning of my journey learning Haskell to start to learn the language. More details about each program will be written below.
## mean.hs
This file contains 3 functions:
- mySum: recursively calculates the sum of all elements in a list
- myLen: recursively calculates the number of elements in a list
- mean: divides the former by the latter to give the mean value in the list
## maxValPropogation.hs
This script takes a list and replaces each element in that list by the maximum value it has seen in the list so far.
For example, [2,1,3,2,1] will give an output of [2,2,3,3,3]
